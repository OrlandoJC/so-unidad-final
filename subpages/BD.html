BASE DE DATOS

MARIA DB

Instalacion y configuracion de la base de datos de MariaDB.

Descripcion

Un servidor de base de datos, también conocido como database server o RDBMS (Relational DataBase Management Systems) en caso de bases de datos relacionales, es un tipo de software de servidor que permiten la organización de la información mediante el uso de tablas, índices y registros.

PASO 1

Se instala la base de datos con el comando apt-get install mariadb-server

  • Ver imagen

PASO 2

Ingresamos al servidor de la base de datos de manera local con el comando mysql –u root –h localhost -p e ingresamos con el password determinado “admin”

  • Ver imagen

PASO 3

Se crea la base de datos con el comando CREATE DATABASE alumnos; después ingresamos a la database con el comando USE alumnos; ahora creamos una tabla llamado alumnos con el comando CREATE TABLE alumnos (nombre VARCHAR(30), edad INT) y se muestra la tabla creada con el comando DESCRIBE alumnos;

  • Ver imagen

PASO 4

Ahora agregamos 2 datos de alumnos dentro la base de datos que creamos con el comando INSERT INTO alumnos VALUES (‘Julissa Aldana’,19); y INSERT INTO alumnos VALUES (‘Gabriel Canche’,20); y mostramos esos datos que insertamos con el comando SELECT*FROM alumnos;

  • Ver imagen

PASO 5

Ahora configuramos el archivo conf de MYSQL con el comando nano /etc/mysql/mariadb.conf.d/50-server-cnf, agregando el nombre de user el cual es “equipo8” y agregamos en bind-adress nuestra ip del servidor en este caso “192.168.0.0” y lo guardamos.

Se instala nmap

PASO 7

Ahora con el comando nmap 192.168.0.0 -p 3306 checamos si el puerto 3306 está abierto para poder utilizar el servicio MYSQL.

PASO 8

Después damos de alta el usuario localhost con el comando alter user ‘root´@´localhost´ identified by “12345”; ahora creamos el usuario equipo8 con el comando create user ‘equipo8´@´%´ identified by “12345”; y de la misma forma creamos un usuario de equipo8 de manera local con el comando create user ‘equipo8´@´localhost´ identified by “12345”;

  • Ver imagen

PASO 9

Concedemos los privilegios al usuario equipo8 para ingresar remotamente con el comando GRANT all privileges ON *.* to ´equipo8´@´%´;

  • Ver imagen

PASO 10

Ahora ingresamos a mariadb y entramos a MYSQL con el comando use mysql , despues para checar los hosts de los usuarios que dimos de alta utilizamos el comando select user, hot from user; y notamos que si se configuro los privilegios del usuario equipo 8 y checamos las bases de datos con el comando show databases;

Se accede desde Windows a la base de datos de Linux Server y vemos las databases disponibles con el comando show databases; y entramos a la databases alumnos con el comando use alumnos y con el comando select*from alumnos; podemos ver los datos que insertamos previamente en nuestro servidor.
