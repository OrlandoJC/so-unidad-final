DHCP

Instalacion y configuracion del servidor DHCP

Descripcion

DHCP significa protocolo de configuración de host dinámico y es un protocolo de red utilizado en redes IP donde un servidor DHCP asigna automáticamente una dirección IP y otra información a cada host en la red para que puedan comunicarse de manera eficiente con otros puntos finales

PASO 1

Primero se instala el dhcp con el comando apt-get install isc-dhcp-server

PASO 2

Ir a adaptador de red y cambiar a adaptador puente Family controller y le damos a permitir todo.

PASO 3

Nos vamos a la opcion de configurar conexiones de red y en la opcion cableada la cambiamos a enp0s8(08:00:27:C5:25:AD)

PASO 4

Nos vamos a la opción de configurar conexiones de red, luego a IPv4 y hacemos las modificaciones pertinentes.

PASO 5

Ahora se activará el servicio dhcp- por lo que primero pondremos el comando ls –l /etc/dhcp.

PASO 6

Luego nos muestra una lista de configuraciones, por lo que primero configuraremos el dhcpd.conf y para eso primero hay que activar las configuraciones con el comando ifup enp0s8

PASO 7

Luego editaremos el archivo dhcp.conf con el comando nano /etc/dhcp/dhcp.conf

PASO 10

Ahora se reinicia el servicio dhcp con el comando systemctl restart isc-dhcp-server y ahora podemos ver el estatus del servidor dhcp con el comando systemctl status isc-dhcp-server
